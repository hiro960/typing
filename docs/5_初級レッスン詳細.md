## レッスン構成
### 初級（子音・母音・簡単な単語）
**目標**: 基本的な子音・母音の配置を覚える

- **レッスン1**: ㅜ, ㅇ, ㅁ + 単語（문, 안, 눈, 나무, 아마, 누나, 만, 바나나）

- **レッスン2**: ㅂ, ㅣ + 単語（입, 밥, 비빔밥, 분, 비누, 방, 위, 반, 아이）

- **レッスン3**: ㄱ, ㅣ + 単語（김밥, 이, 가위, 위, 귀, 입, 누구, 비누, 아기）

- **レッスン4**: ㄴ, ㄷ + 単語（다, 도, 두, 나, 노, 누나, 다리, 도시, 두부, 나무）

- **レッスン5**: ㄹ, ㅅ + 単語（사, 소, 수, 라, 로, 수도, 사다, 소리, 사람, 수박）

- **レッスン6**: ㅈ, ㅓ + 単語（저, 조, 주, 자, 지도, 저기, 주소, 자리, 저녁, 조금）

- **レッスン7**: ㅎ, ㅗ + 単語（하, 호, 히, 오, 하나, 호수, 오다, 하루, 호랑이, 오이）

- **レッスン8**: ㅋ, ㅌ, ㅍ + 単語（카, 타, 파, 코, 토, 포도, 커피, 타다, 파다, 토끼）

- **レッスン9**: ㅊ + 単語（차, 처, 초, 추, 치다, 차다, 처음, 초등학생, 추다, 치킨）

- **レッスン10**: ㅐ, ㅔ + 単語（개, 게, 배, 베, 새, 세, 애기, 개미, 배추, 세수）

- **レッスン11**: ㅡ + 単語（느, 드, 르, 스, 그, 크다, 슬프다, 그림, 스키, 쓰다）

- **レッスン12**: 濃音 ㄲ, ㄸ, ㅃ + 単語（까, 빠, 따, 꼬리, 뿌리, 딸기, 까치, 빠르다, 또）
  - 濃音は舌の緊張を伴う発音。普通の子音より強く発音される

- **レッスン13**: 濃音 ㅆ, ㅉ + 単語（싸, 짜, 쓰다, 씨, 짜다, 쌀, 찌다, 쏘다, 짜증）

- **レッスン14**: 複合母音 ㅑ, ㅕ, ㅛ, ㅠ + 単語（야, 여, 요, 유, 야구, 여기, 요리, 유리, 약, 연필）
  - 複合母音は二重母音。「ㅣ」の音が含まれる

- **レッスン15**: 複合母音 ㅘ, ㅝ, ㅢ, ㅚ, ㅟ + 単語（와, 왜, 워, 의, 외국, 왜냐하면, 귀, 취미, 웨딩）

- **レッスン16**: パッチム基礎（終声）+ 単語（밥, 집, 책, 옷, 밖, 꽃, 산, 달, 물, 불）
  - パッチム（받침）は韓国語の終声。文字の下に来る子音

- **レッスン17**: パッチム応用 + 単語（말, 얼굴, 일, 발, 날씨, 잘, 밤, 공, 강, 맛）

- **レッスン18**: 二重パッチム ㄳ, ㄺ, ㄻ + 単語（값, 넓다, 삶, 읽다, 닭, 흙, 앉다, 옳다）
  - 二重パッチムは2つの子音が組み合わさった終声。発音は1つの音になることが多い

- **レッスン19**: 二重パッチム ㄼ, ㄽ, ㄾ, ㄿ, ㅀ, ㅄ + 単語（넓다, 밟다, 핥다, 읊다, 없다, 값다, 짧다, 많다）

- **レッスン20**: 総復習 - 簡単な文章（안녕하세요, 감사합니다, 저는 학생입니다, 만나서 반갑습니다, 잘 부탁합니다）
  - これまでに学習した文字を組み合わせた日常会話フレーズ
