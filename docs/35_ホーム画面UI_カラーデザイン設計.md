# ホーム画面 UI/UX カラーデザイン設計

## 概要

ホーム画面のUI/UXを改善し、ティールブルー系の清潔感のある配色を採用。
ターゲットユーザーである韓国語学習に興味のある女性向けに、優しく見やすいデザインを実現。

## ターゲットユーザー

- 韓国語に興味のある女性
- 20〜40代を中心とした学習者
- 清潔感・信頼感のあるデザインを好むユーザー

## デザインコンセプト

### 参考アプリ

**Awarefy** - AIメンタルパートナーアプリ
- ティールブルーを基調とした清潔感のあるデザイン
- 白基調の背景に、ティールのアクセントカラー
- 女性ユーザーに人気のある配色

### デザイン方針

1. **清潔感** - 白基調の背景で視認性を確保
2. **信頼感** - ティールブルーの落ち着いた色調
3. **統一感** - 全体的にティール系のグラデーションで統一
4. **視認性** - コントラストを確保し、テキストを読みやすく

## レイアウト変更

### Before（変更前）

```
ヘッダー
  ↓
グラデーション装飾（120px）
  ↓
週間進捗カード
  ↓
Quick Tools
  ↓
タイピング練習 Hero
  ↓
学ぶセクション（3アイコン）
  ↓
遊ぶセクション（3アイコン）
  ↓
調べるセクション（2アイコン）
  ↓
AI先生
```

### After（変更後）

```
ヘッダー
  ↓
Quick Tools
  ↓
タイピング練習 Hero
  ↓
機能グリッド（2行×4列、8アイコン）
  ↓
週間進捗カード
  ↓
AI先生
```

### 主な変更点

1. **グラデーション装飾を削除** - 余分なスペースを解消
2. **カテゴリ廃止** - 「学ぶ/遊ぶ/調べる」の3セクションを統合
3. **グリッドレイアウト** - 2行×4列のコンパクトなグリッドに
4. **週間進捗の位置変更** - グリッドの下、AI先生の上に移動

## デザインの効果

1. **視認性向上** - 白基調の背景でコンテンツが見やすい
2. **統一感** - ティール系で全体をまとめ、一貫性のあるデザイン
3. **省スペース** - カテゴリ統合により縦スクロール量を削減
4. **モダンな印象** - Iconsaxアイコンで洗練されたデザイン
5. **女性ユーザーに好印象** - 清潔感のあるティールブルーは女性に人気
