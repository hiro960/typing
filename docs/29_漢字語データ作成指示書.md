# 漢字語（熟語）データ作成指示書

## 概要

漢字語辞典の漢字語（熟語）データ（約5,000語）を作成します。この作業を5人で分担して行います。

## JSONファイル構造

### 保存場所
```
application/assets/hanja/words/
├── words_daily.json         # 担当5-1: 日常生活
├── words_education.json     # 担当5-2: 教育・学問
├── words_society.json       # 担当5-3: 社会・政治・経済
├── words_nature.json        # 担当5-4: 自然・科学・医療
└── words_culture.json       # 担当5-5: 文化・芸術・その他
```

### JSONテンプレート

```json
{
  "version": "1.0.0",
  "category": "daily",
  "categoryName": "日常生活",
  "totalWords": 1000,
  "words": [
    {
      "id": "word_0001",
      "word": "가족",
      "hanja": "家族",
      "meaning": "家族",
      "pronunciation": "가족",
      "partOfSpeech": "명사",
      "level": "basic",
      "frequency": "high",
      "components": [
        {"character": "家", "korean": "가", "meaning": "家"},
        {"character": "族", "korean": "족", "meaning": "族、一族"}
      ],
      "examples": [
        {
          "sentence": "우리 가족은 다섯 명이에요.",
          "meaning": "私の家族は5人です。"
        }
      ],
      "relatedWords": ["가정", "가문", "친족"],
      "antonyms": [],
      "synonyms": ["식구"]
    }
  ]
}
```

### フィールド説明

| フィールド | 必須 | 説明 | 例 |
|-----------|------|------|-----|
| id | ○ | 一意のID（word_XXXX形式） | "word_0001" |
| word | ○ | 漢字語（ハングル） | "가족" |
| hanja | ○ | 漢字表記 | "家族" |
| meaning | ○ | 意味（日本語） | "家族" |
| pronunciation | ○ | 発音（発音変化がある場合） | "가족" |
| partOfSpeech | ○ | 品詞 | "명사" |
| level | ○ | レベル | "basic" |
| frequency | △ | 頻度 | "high" |
| components | ○ | 構成漢字の分解 | [...] |
| examples | ○ | 例文（最低1つ） | [...] |
| relatedWords | △ | 関連語 | ["가정", "가문"] |
| antonyms | △ | 反意語 | [] |
| synonyms | △ | 類義語 | ["식구"] |

### 品詞（partOfSpeech）の種類

| 韓国語 | 日本語 |
|--------|--------|
| 명사 | 名詞 |
| 동사 | 動詞 |
| 형용사 | 形容詞 |
| 부사 | 副詞 |
| 관형사 | 冠形詞 |

## 漢字表記の注意事項

### 韓国で使用される漢字（繁体字ベース）

韓国語の漢字語は、日本の常用漢字とは異なる字体を使う場合があります。

| 日本 | 韓国 | ハングル |
|------|------|---------|
| 学 | 學 | 학 |
| 経 | 經 | 경 |
| 会 | 會 | 회 |
| 国 | 國 | 국 |
| 医 | 醫 | 의 |
| 芸 | 藝 | 예 |
| 歴 | 歷 | 역 |

**可能であれば韓国式の漢字（繁体字）を使用してください。**

## 発音変化の記録

発音変化がある場合は`pronunciation`フィールドに記録します。

```json
{
  "word": "학교",
  "hanja": "學校",
  "pronunciation": "학꾜"
}
```

```json
{
  "word": "십만",
  "hanja": "十萬",
  "pronunciation": "심만"
}
```

---

## components（構成漢字）の記録

各漢字語を構成する漢字を分解して記録します。

```json
{
  "word": "대한민국",
  "hanja": "大韓民國",
  "components": [
    {"character": "大", "korean": "대", "meaning": "大きい"},
    {"character": "韓", "korean": "한", "meaning": "韓"},
    {"character": "民", "korean": "민", "meaning": "民"},
    {"character": "國", "korean": "국", "meaning": "国"}
  ]
}
```

## 品質チェックリスト

各担当者は提出前に以下を確認してください：

- [ ] JSON構文エラーがないか
- [ ] 必須フィールドが全て入っているか
- [ ] IDが重複していないか
- [ ] IDが指定範囲内か
- [ ] 漢字表記が正しいか
- [ ] 例文が最低1つあるか
- [ ] componentsが正しく分解されているか
