# 単漢字データ作成指示書

## 概要

漢字語辞典の単漢字データ（約3,586字）を作成します。この作業を4人で分担して行います。

## データソース

**主要ソース**: https://www.penguin99.com/08_kanji/kanji_ichiran_jp_kr.html

このページから日本語音読み・韓国語漢字音・対応漢字を抽出し、JSONファイルを作成します。

## JSONファイル構造

### 保存場所
```
application/assets/hanja/characters/
├── characters_a_ka.json     # 担当1: ア行〜カ行
├── characters_sa_ta.json    # 担当2: サ行〜タ行
├── characters_na_ha.json    # 担当3: ナ行〜ハ行
└── characters_ma_wa.json    # 担当4: マ行〜ワ行
```

### JSONテンプレート

```json
{
  "version": "1.0.0",
  "range": "ア行〜カ行",
  "totalCharacters": 1200,
  "characters": [
    {
      "id": "hanja_0001",
      "character": "愛",
      "unicode": "U+611B",
      "korean": "애",
      "japaneseOn": ["アイ"],
      "japaneseKun": ["いと.しい", "まな"],
      "meaning": "愛、愛する",
      "strokeCount": 13,
      "radical": "心",
      "radicalKorean": "심",
      "level": "basic",
      "frequency": "high",
      "relatedWords": ["사랑", "애정", "연애"]
    }
  ]
}
```

### フィールド説明

| フィールド | 必須 | 説明 | 例 |
|-----------|------|------|-----|
| id | ○ | 一意のID（hanja_XXXX形式） | "hanja_0001" |
| character | ○ | 漢字1文字 | "愛" |
| unicode | ○ | Unicodeコードポイント | "U+611B" |
| korean | ○ | 韓国語読み（ハングル） | "애" |
| japaneseOn | ○ | 日本語音読み（配列） | ["アイ"] |
| japaneseKun | △ | 日本語訓読み（配列、あれば） | ["いと.しい"] |
| meaning | ○ | 意味（日本語） | "愛、愛する" |
| strokeCount | △ | 画数 | 13 |
| radical | △ | 部首（漢字） | "心" |
| radicalKorean | △ | 部首（韓国語名） | "심" |
| level | ○ | レベル（basic/intermediate/advanced） | "basic" |
| frequency | △ | 頻度（high/medium/low） | "high" |
| relatedWords | △ | 関連する漢字語（配列） | ["사랑", "애정"] |

## 担当1: ア行〜カ行

### 担当範囲
- **日本語音読み**: ア、イ、ウ、エ、オ、カ、キ、ク、ケ、コ
- **推定字数**: 約1,200字
- **ID範囲**: hanja_0001 〜 hanja_1200

### 出力ファイル
`application/assets/hanja/characters/characters_a_ka.json`

### 作業手順

1. https://www.penguin99.com/08_kanji/kanji_ichiran_jp_kr.html を開く
2. ア行〜カ行の漢字を抽出
3. 各漢字について以下を記録:
   - 漢字
   - 韓国語読み（ハングル）
   - 日本語音読み
4. 追加情報を補完:
   - 意味
   - 画数（可能な場合）
   - 部首（可能な場合）
5. JSONファイルを作成

### サンプルデータ（ア行）

```json
{
  "id": "hanja_0001",
  "character": "亜",
  "unicode": "U+4E9C",
  "korean": "아",
  "japaneseOn": ["ア"],
  "japaneseKun": [],
  "meaning": "亜、次ぐ",
  "strokeCount": 7,
  "radical": "二",
  "radicalKorean": "이",
  "level": "intermediate",
  "frequency": "medium",
  "relatedWords": ["아시아", "아연"]
}
```

## 同音異字の処理

同じ韓国語読みで異なる漢字がある場合、それぞれ別のエントリとして作成します。

```json
// 「사」と読む漢字が複数ある場合
{
  "id": "hanja_1201",
  "character": "査",
  "korean": "사",
  ...
},
{
  "id": "hanja_1202",
  "character": "士",
  "korean": "사",
  ...
},
{
  "id": "hanja_1203",
  "character": "事",
  "korean": "사",
  ...
}
```

---

## 複数読みの処理

一つの漢字に複数の韓国語読みがある場合、japaneseOnの関連する読みとセットで記録します。

```json
{
  "id": "hanja_0500",
  "character": "行",
  "korean": "행",
  "koreanAlt": ["항"],
  "japaneseOn": ["コウ", "ギョウ", "アン"],
  "meaning": "行く、行、行う",
  ...
}
```

## level（レベル）の基準

| レベル | 説明 | 例 |
|--------|------|-----|
| basic | 基礎漢字、TOPIK初級相当 | 人、山、水、火 |
| intermediate | 中級漢字、日常会話で使用 | 経済、政治、文化 |
| advanced | 上級漢字、専門用語 | 哲学、弁証、概念 |

---

## frequency（頻度）の基準

| 頻度 | 説明 |
|------|------|
| high | 非常によく使われる（上位500字程度） |
| medium | 普通に使われる |
| low | あまり使われない、専門的 |

## 品質チェックリスト

各担当者は提出前に以下を確認してください：

- [ ] JSON構文エラーがないか（JSON Validatorで確認）
- [ ] 必須フィールド（id, character, unicode, korean, japaneseOn, meaning, level）が全て入っているか
- [ ] IDが重複していないか
- [ ] IDが指定範囲内か
- [ ] 漢字と韓国語読みが正しく対応しているか
- [ ] Unicodeが正しいか

## Unicode取得方法

Pythonでの取得例：
```python
char = "愛"
print(f"U+{ord(char):04X}")  # U+611B
```

JavaScriptでの取得例：
```javascript
const char = "愛";
console.log(`U+${char.charCodeAt(0).toString(16).toUpperCase()}`);
```
